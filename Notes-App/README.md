
# **Responsive Notes App**

A simple and responsive notes app built using HTML, CSS, and JavaScript.
You can create notes, delete notes, and all notes are saved automatically using **localStorage**.

https://notes-app-orpin-five.vercel.app/

## **Features**

* Add new notes
* Delete notes
* Notes are saved in localStorage
* Responsive and easy-to-use interface

## **How to Use**

1. Download or clone the project
2. Open `index.html` in your browser
3. Click **Create Note** to add a note
4. Your notes will stay even after refreshing the page

## **Files**

* `index.html` – Main structure
* `style.css` – Styles and layout
* `script.js` – App logic and localStorage
